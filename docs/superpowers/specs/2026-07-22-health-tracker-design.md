# 健康追踪 App 设计规格

**日期：** 2026-07-22
**状态：** 设计确认，待实施

## 概述

一个纯客户端 PWA（渐进式 Web App），运行在手机浏览器中，数据全部存储在手机本地（IndexedDB），不经过任何服务器或云端。每日通过问答引导方式记录五大维度的健康数据，支持随时查看/修改历史记录，并一键导出为 Markdown 文件。

## 产品形态

- 纯前端 PWA，单 HTML 文件
- 通过 Service Worker 支持离线访问
- 添加到 iPhone 主屏幕获得类似原生 App 的入口体验
- 无需任何后端服务、无需注册、无网络依赖
- 优先适配 iOS Safari 竖屏，支持 safe area

## 五大维度（L1 顶层维度）

每天对每个维度打出 1-10 的总分。

### 1. 生理躯体健康（底层硬件基础）

**睡眠维度**
- 入睡难度：单选（失眠、困难2h内入睡、中等1h内入睡、简单0.5h内入睡）
- 总有效睡眠时长：数字选择（1-24小时）
- 睡眠深浅：单选（深、浅）
- 睡眠质量：单选（半夜易醒、多梦）
- 晨起疲惫感：单选（有、无）
- 作息节律稳定性：单选（坚持早睡一周7天、早睡晚睡相间、持续晚睡）

**饮食与消化**
- 食欲：单选（好、一般、差）
- 早餐：多选（没吃、馒头、包子、水煮蛋、方便面、小米粥、白粥、汤泡饭、早餐面条）
- 午餐：多选（辣炒卤牛肉、辣炒鱿鱼丝、辣炒卤猪肉、青椒炒肉丝、小鸡炖蘑菇、鹌鹑蛋红烧肉、红烧鱼、青椒炒鸡蛋干、炒生菜、炒空心菜、炒油麦菜、青菜炒腐竹、山药炒木耳、青椒土豆丝、牛肉丸生菜汤、番茄鸡蛋汤、紫菜鸡蛋汤、冬瓜排骨汤、玉米排骨汤、萝卜排骨汤、香菇鸡汤、鱼汤、米饭、馒头、拌面、牛肉丸粿条、江西米粉、水饺、皮蛋瘦肉粥、白粥）
- 晚餐：多选（同午餐选项）
- 饮品：多选（无、咖啡、奶茶、牛奶）
- 三餐规律：多选（没吃早饭、午餐过晚、没吃晚饭）
- 饮食情况：单选（暴饮暴食、节食）
- 饮水情况：单选（正常、少）
- 大便状态：单选（窜稀、便秘、没拉）

**运动与体能**
- 运动记录：可追加多组，每组 = 运动类型 + 活动时长
  - 运动类型：多选（八段锦、金刚功、长寿功、跑步、散步、骑自行车、爬山、举哑铃）
  - 活动时长：单选（10min、20min、30min、1h、2h以上）
- 久坐时长：单选（久坐轻度<4h、久坐中度4~6h、久坐重度≥6h）

**体征与体感**
- 气血状态：单选（平和、偏虚、阻滞、郁热或虚热）
- 体感状态：多选（畏寒、上火、乏力、头晕、腰疼、肩膀酸痛、腹痛、胃痛、肚子痛、肚子右上腹痛）

**就医与用药**
- 医院：填写框
- 科室：填写框
- 就诊原因：填写框
- 用药：填写框
- 用药原因：填写框

### 2. 心理情绪健康（内在感受中枢）

**情绪基调（当下主观感受）**
- 多选标签（平静、愉悦、低落、烦躁、焦虑、空虚、麻木）

**心智状态**
- 单选（内心杂念、反刍思考、过度纠结、消费冲动、争吵冲动）
- 每选一个心智状态，需填写原因（填写框）

**压力与恢复力**
- 当日压力来源：填写框
- 压力耐受度：单选（高、中、低）
- 自我安抚能力：单选（高、中、低）
- 安抚行为：填写框
- 情绪修复速度：单选（快、中、慢）

**内在接纳度**
- 是否自我否定：是/否，原因：填写框
- 对当下状态是否包容：是/否，原因：填写框

### 3. 时间与行为健康（心智自律层）

**任务执行状态**
- 计划完成率：数字选择器（1%-100%）
- 拖延程度：单选（无、轻微、中度、重度）
- 启动阻力：填写框
- 能否进入心流：单选（容易 / 困难）
- 重要事务是否持续搁置：单选（是、否）

**时间分配与成瘾行为**
- 短视频/被动娱乐耗时：数字选择器（1-24）h
- 碎片化刷屏频次：数字选择器（1-100）次

**节律自控**
- 是否出现报复性休闲、逃避式消遣：单选（是、否）
- 是否出现忙时自律、空闲摆烂：单选（是、否）

**行为止损能力**
- 察觉到虚度之后，能否主动中断不良行为：单选（是、否）
- 及时调整状态：填写框（填写行为）

### 4. 人际亲密健康（外部互动系统）

**亲密关系沟通（区分男朋友和父母两个角色）**
- 沟通顺畅度：单选（顺畅、勉强、困难）— 每个角色分别选
- 矛盾冲突：填写框
- 理解感：单选（高理解感 / 缺少理解感）
- 期待落差：单选（存在期待落差 / 期待基本契合）
- 争吵与情绪隔阂：多选（发生争吵 / 存在情绪隔阂 / 关系融洽无隔阂）

**社交平衡**
- 有效社交时长：数字选择器（1-24）h
- 社交消耗感：单选（大、中、小）
- 独处：单选（感到安宁、空虚难耐）

**边界感受**
- 是否容易被他人情绪影响、难以拒绝他人、过度妥协：单选（有、无），原因：填写框

**情感支持获取**
- 情绪低落时是否拥有可靠倾诉渠道：单选（有、无），渠道：填写框

### 5. 精神目标健康（顶层意义坐标）

**目标清晰度**
- 是否拥有短期事务目标：单选（是、否）
- 是否拥有长期发展方向：单选（是、否）
- 目标是否模糊摇摆：单选（是、否）

**价值感知**
- 一天结束是否产生"虚度、浪费时间"的感受：单选（是、否）
- 做事能否获得充实感：单选（是、否）

**内在秩序感**
- 生活是否有稳定主线：单选（是、否）
- 容易陷入漫无目的、被动消磨时光：单选（是、否）

**长期动力**
- 单选（持续保有成长意愿 / 持续躺平、失去行动渴望）

## 每日问答流程

自由选择维度填写，每个维度格式如下：
1. 显示该维度名称和定义
2. 依次展示每个子项的结构化控件（单选、多选、数字选择、填写框）
3. 所有子项填完后，显示该维度总分输入（1-10 数字选择器）
4. 完成后自动进入下一维度

## 数据存储

使用 IndexedDB。

## 关键交互特性

### 自动保存
离开输入框或选择控件时自动写入 IndexedDB。无需手动保存。

### 历史记录查看与修改
- 主页提供日历视图或日期列表
- 点击任意日期进入查看/编辑模式
- 所有内容均可修改，修改后自动保存

### 完成标记
- 当日所有 5 个维度总分填写完毕时，标记当日记录为 completed: true

## 导出功能

导出为 Markdown 文件，格式为结构化数据转 MD。导出选项：
- 导出单天
- 导出本周
- 导出全部

## 技术栈

- HTML/CSS/JS：单页面应用，内联在一个 HTML 文件中
- 存储：IndexedDB（原生浏览器 API）
- 离线支持：Service Worker
- PWA 清单：manifest.json
- 兼容性：优先适配 iOS Safari（iPhone 竖屏），safe area 适配
- 无任何第三方依赖

## UI 布局

- 手机竖屏全屏，单列滚动
- 顶部：当前日期、5 维度进度指示器
- 按维度分块，结构化控件展示
- 底部固定导航：主页（日历）、导出

## 复盘分析（非 App 功能）

App 仅记录和导出。用户将导出 MD 文件提供给外部 AI（ChatGPT/Claude）进行交叉分析。
